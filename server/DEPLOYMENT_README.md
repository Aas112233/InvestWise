# Deployment Artifacts

This folder contains the build artifacts generated on 2026-02-14.

## Files
1. **`dist/`**: The folder containing the production build of the frontend application.
   - Includes `.htaccess` for Apache configuration (InfinityFree).
   - Includes optimized assets.

2. **`frontend-deployment.zip`**: A compressed archive of the `dist/` folder, ready for upload.

## Quick Start
1. Take `frontend-deployment.zip`.
2. Extract contents or upload directly to your web server (e.g., InfinityFree htdocs).
3. Ensure `.htaccess` is included in the root of your web directory.

For detailed deployment steps, see [DEPLOYMENT.md](./DEPLOYMENT.md).
