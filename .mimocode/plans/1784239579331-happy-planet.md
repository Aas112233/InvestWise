# Goals Screen Redesign Plan

## Overview
Redesign the Goals screen (`client/components/Goals.tsx`) to provide a more engaging, visually rich experience while maintaining consistency with the app's design system (brand color `#BFF300`, rounded cards, `font-black` typography, dark mode support).

## Current State
The existing Goals screen has:
- Header with title/subtitle + "Add Goal" button
- 4 stat cards (Total Goals, Achieved, In Progress, Total Target)
- 3-column card grid with left color stripe, progress bar, financial details, deadline
- Basic modal form for create/edit

## Redesign Goals
1. **More visually dynamic** - Add a hero summary section with a large circular progress ring
2. **Better goal cards** - Richer cards with milestone markers, urgency indicators, and better progress visualization
3. **Filter & search** - Add ability to filter by type/status
4. **Improved empty state** - More engaging illustration and call-to-action
5. **Responsive** - Better mobile card layout
6. **Maintain** - All existing functionality (CRUD, translations, dark mode, form validation)

---

## Changes

### File: `client/components/Goals.tsx`

#### 1. Hero Summary Section
Replace the plain stats row with a visually prominent hero section:
- **Left side**: Large circular SVG progress ring showing overall completion % (across all goals)
- **Right side**: 3 mini stat pills (Achieved / In Progress / Cancelled counts) and total target amount
- Uses brand color gradient for the ring, consistent with the app's lime-green aesthetic

#### 2. Filter Bar
Add a horizontal filter bar below the hero section:
- Pill-style toggle buttons for goal types: All | Savings | Investment | Other
- Pill-style toggle for status: All | In Progress | Achieved | Cancelled
- Uses existing `Button` component with `ghost` variant for active state styling

#### 3. Goal Cards Redesign
Enhance each card:
- **Circular mini progress ring** (instead of linear progress bar) in the card header area
- **Urgency indicator**: Color-coded dot/badge if deadline is within 7 days, overdue, etc.
- **Milestone markers**: Show 25%, 50%, 75%, 100% thresholds with subtle tick marks
- **Linked project badge** if `linkedProject` is set
- **Relative deadline**: "5 days left" / "3 days overdue" prominently displayed
- Better spacing, consistent card hover effects (`hover:-translate-y-2 hover:shadow-xl`)
- Keep the left color stripe by type (Savings=emerald, Investment=purple, Other=blue)

#### 4. Empty State
Use the app's `EmptyState` component with type `data` for a polished empty state with illustration and CTA button.

#### 5. Modal Form Enhancements
- Add a description `FormTextarea` field (currently exists in form state but not in the UI)
- Keep existing form structure, just ensure description field is visible
